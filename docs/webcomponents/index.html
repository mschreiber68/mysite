Lately I have been interested in Web Components.

Building Blocks

These components aim to encapsulate the javascript for a general user interaction. They are 100% customizable and styleable. They are most useful as building blocks for more advanced components.

Toggle

A Toggle is made up a binary state and a toggle element. When the element is interacted with, the state inverts. Toggles can be used to build things like dropdowns and tooltips.

Mux

A Mux, or Multiplexer, consists of a set of selections to choose from. Only one selection can be active at a time. Muxes can be used to build things like tabs and accordions.
